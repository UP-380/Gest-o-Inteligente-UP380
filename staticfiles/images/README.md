# Pasta de Imagens

## Logo do +Compliance

Para adicionar o logo do sistema, coloque o arquivo de imagem na pasta `static/images/` com o nome `logo.png`.

### Especificações recomendadas:
- **Formato**: PNG (com transparência) ou SVG
- **Tamanho**: 80x80 pixels (ou múltiplos para alta resolução)
- **Nome do arquivo**: `logo.png`

### Como adicionar:
1. Coloque o arquivo `logo.png` nesta pasta
2. O sistema automaticamente carregará a imagem no template de login
3. Se não houver imagem, será exibido um placeholder azul com a letra "C"

### Estrutura esperada:
```
static/
└── images/
    └── logo.png  <- Seu logo aqui
```
